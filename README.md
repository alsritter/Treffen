## Treffen

Treffen is an employee management system, which contains a role system and uses Redis as a database cache

Server side：
SpringBoot, SpringSecurity, Redis, MyBatisPlus, JWT, Swagger2

Front side：
Vue3.0, Typescript, Less, Vue-Cli4
